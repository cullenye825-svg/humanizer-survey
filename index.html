Reading judgement

Which one reads like a person wrote it?

You'll see a series of short passage pairs. For each, pick the one that better answers the question shown — usually which reads more like a real person wrote it. A few ask which reads more like a particular kind of writing instead; the question is always at the top.

There are no right answers. Go with your first instinct — you don't need to re-read. It takes about fifteen minutes, and nothing is sent anywhere: at the end you'll get a short code to send back.
